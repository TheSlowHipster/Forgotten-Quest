Directions : 

0.) Download a java JDK

1.) Download the repository as a .zip file
2.) Extract the repository to the desired location
3.) Open the command line
4.) From the command line open the directory
5.) Compile the files using 'javac Game.java'
6.) Run the game using 'java Game'
7.) Enjoy
